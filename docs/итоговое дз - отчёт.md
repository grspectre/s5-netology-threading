# Actuator + Prometheus + Grafana

## Что подключено
- spring-boot-starter-actuator
- micrometer-registry-prometheus

## Actuator endpoints
Доступны:
- /actuator/health
- /actuator/info
- /actuator/metrics (в т.ч. http.server.requests)
- /actuator/prometheus
- /actuator/threaddump

## Как запустить
1) Запуск приложения:
   - mvn spring-boot:run
2) Запуск мониторинга:
   - docker compose up -d
3) URLs:
   - App: http://localhost:8080/
   - Prometheus: http://localhost:9090
   - Grafana: http://localhost:3000 (admin/admin)
